# 观潮 Codex Skill

这个仓库保存 Codex Skill：`tech-news-to-video`。

触发语：`开始观潮` 或 `开启观潮`。

用途：从最新科技前沿新闻中筛选高热选题，用“把科技讲成局”的观潮方法，拆解技术权力迁移、地缘政治、金融定价、未来路径和破局逻辑，并输出 15 分钟中文视频包。需要制作视频时，目标交付为可下载 MP4。

## 使用方式

在另一台电脑使用时，把本仓库中的 `skills/tech-news-to-video` 文件夹复制或同步到 Codex 的 skills 目录，然后在对话中说：

`开始观潮`

## 目录

- `skills/tech-news-to-video/SKILL.md`: Skill 主说明
- `skills/tech-news-to-video/agents/openai.yaml`: Skill 显示与默认提示配置
- `skills/tech-news-to-video/references/platform-review.md`: 抖音、小红书、视频号审核清单
- `skills/tech-news-to-video/references/video-production.md`: 视频制作标准

## 注意

该 Skill 要求联网核查最新新闻。涉及视频制作时，优先调用可用的视频工具，例如 HyperFrames by HeyGen。
